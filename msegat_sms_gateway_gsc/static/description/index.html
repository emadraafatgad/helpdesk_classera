Create & Configure Msegat Account in Odoo

Following steps to you can create and configure Msegat account in Odoo, 

1. You have required Msegat account if have not then the following link to you can create account.
	> https://www.msegat.com/index.php
	
2. After in Odoo, Goto Msegat SMS Gateway App > Configurations > Msegat Accounts Menu.

3. Create new account and fill-up all required informations,
	* API Credentials Informations
	3.1 UserName: Here you have to enter your login username which you are use during login into Msegat website.
	3.2 API Key: API key you can get from Msegat.
	3.3 Sender ID/TAG Name: Click on "Fetch Sender ID/TAG Name" button to automatically fetch all Tag Names from Msegat.
	3.4 Is Default SMS Account?: If you enabled this option then this account set as default account.
		> If you have enabled Advanced Features then while confirming Sale Order or Delivery Order time system will pick default SMS account and send SMS to customer.
	
	* Test Connection and Confirm Msegat Account in Odoo
	3.5 Click on "Test Connection" button to system will check Odoo is successfully connected or not.
	3.6 Click on "Confirm Account" button to confirm your Odoo Msegat Account.
	
|	
	* Test Send SMS
	3.7 Click on "Send Test SMS" button to system will send test SMS to "Test Connection Mobile Number".
|	
	** Get Msegat Account Balance
	3.8 Click on "Fetch Balance" button to system will fetch current balance from Msegat.

Create and Configure Msegat SMS Groups in Odoo

Following steps to you can create and configure Msegat SMS Groups in Odoo, 

1. Goto Msegat SMS Gateway App > Configurations > SMS Groups Menu.
2. Create new SMS Group and fill-up all required informations,
> Group Name, Select recipients

Create and Configure Msegat SMS Templates in Odoo

Following steps to you can create and configure Msegat SMS Templates in Odoo, 

1. Goto Msegat SMS Gateway App > Configurations > SMS Template Menu.
2. Create new SMS Template and fill-up all required informations,
> Template Name and Message.

Send SMS to Recipients

Following steps to you can send SMS from Odoo, 

1. Goto Msegat SMS Gateway App > SMS Menu > Send SMS Menu.
2. Create new SMS record and 
	2.1 Select SMS Account for sending message.
	2.3 Send SMS To:
		2.3.1 Contact: You can send SMS to single contact.
		2.3.2 Multiple Contact: You can send SMS to multiple contacts.
		2.3.3 SMS Group: You can send message to group(which have multiple contact).
		2.3.4 Mobile: You can send SMS to direct mobile number if not exist in Odoo.
	2.4 SMS Template: You can select predefine SMS template so don't need to write same message multiple time.
	2.5 Message: You can write your message for send to your contacts.
3. Click on "Send Message" button to send message to your contacts.

Msegat Informations: After sending messages to Contacts/Customers, You will get the information about How many messages sent Successfully / Failed.

Log History: Here you will get the full log information about sent messages.

Confirm Sale/Website Order to Auto Send SMS

Following steps to you can auto send SMS while confirm sale order, 

1. Goto Configuration > Msegat Account. Open any Msegat Account which for you want to enable this feature.
	1.1 Is Default SMS Account?: If you want to send SMS then you have to enable this option for default auto pick this account while confirmation of the sale order.
	1.2 There is "Confirm Order to Send SMS?" option need to enable.
	1.3 SMS Template: Select SMS Template.
	>> For template create, goto Configuration >> SMS Template.
	>> Then, Create SMS Template and fillup following informations,
		>> Template Name: i.e Sales Order: Confirmation SMS
		>> Applies To: Select Sale Order
		>> Message: Here type message as you like. Here you can access all Sales Order Model's and Other fields which are linked with Sale Order Model.
		Scroll down to you can see the example SMS Template.
		>> Preview Magic Button: Here you can preview your wrote message for existing sale order. 
		Note: This preview features to not send message to customer, It's only for preview.  
2. After, when you confirm sales/website order that time auto send SMS to customer.

Validate Delivery/Picking to Auto Send SMS

Following steps to you can auto send SMS while validate delivery, 

1. Goto Configuration > Msegat Account. Open any Msegat Account which for you want to enable this feature.
	1.1 Is Default SMS Account?: If you want to send SMS then you have to enable this option for default auto pick this account while confirmation of the sale order.
	1.2 Is SMS Account?: Required to enable this option for sending message to customer.
	1.3. There is "Validate Delivery to Send SMS?" option need to enable.
	1.4 SMS Template: Select SMS Template.
	>> For template create, goto Configuration >> SMS Template.
	>> Then, Create SMS Template and fillup following informations,
		>> Template Name: i.e Delivery: Send by SMS Text Message
		>> Applies To: Select Transfer
		>> Message: Here type message as you like. Here you can access all Delivery Model's and Other fields which are linked with Delivery Model.
		Scroll down to you can see the example SMS Template.
		>> Preview Magic Button: Here you can preview your wrote message for existing delivery orders. 
		Note: This preview features to not send message to customer, It's only for preview.  
2. After, when you validate delivery order that time auto send sms to customer.
